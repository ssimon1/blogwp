== Description ==

A plugin made by spiraclethemes.com to extends its free themes features by adding functionality to import demo data content in just a click.


How to use this Spiraclethemes site library ?

1. While installing free themes by spiraclethemes do kindly install & activate our plugin " Spiraclethemes Site Library " or you can do this manually too.
2. To import demo contents, please go to Appearance > Import Demo Data. 
3. Choose your demo by checking live preview and then click import button to import the demo.
4. Feel free to contact spiraclethemes support team at ( http://support.spiraclethemes.com/ ) if you need any assistance.

== Installation ==

*** From your WordPress dashboard ***

1. Visit 'Plugins > Add New',
2. Search for 'Spiraclethemes Site Library' and install the plugin.
3. Activate 'Spiraclethemes Site Library' from your Plugins page.


== Frequently Asked Questions ==

= Does this plugin works with any WordPress theme =

No, This plugin will only work with the themes developed by [Spiracle Themes](https://spiraclethemes.com/).
